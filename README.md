## 🎀 Functions 
- ೃ⁀➷ **Write & Save Notes**  
  Type a note and hit enter to add it to your notepad. You can create multiple notes and view them all on display.
- ೃ⁀➷ **Multiple Notes Display**  
  All your saved notes are shown together so you can see everything you've jotted down at a glance.
- ೃ⁀➷ **Live Weather Widget**  
  Pulls real-time weather data from the [Open-Meteo API](https://open-meteo.com/) so you can check the weather right from your notepad.(From your location)
- ೃ⁀➷ **Live Clock**  
  Displays the current time, updating in real time.
